# Clean Bros Chrudim - Landing Page

Profesionální úklid domácností a firem v Chrudimi a okolí.

## Technologie

- Čisté HTML5
- Moderní CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Responzivní design (mobile-first)

## Jak nasadit na GitHub Pages

1. Vytvořte nový repository na GitHub.com (např. `clean-bros-chrudim`)
2. Nahrajte soubor `index.html` do repository
3. V nastavení repository (Settings → Pages):
   - Source: Deploy from a branch
   - Branch: main (nebo master) a složka `/ (root)`
   - Save
4. Váš web bude dostupný na: `https://vasusername.github.io/clean-bros-chrudim/`

## Alternativa - Netlify (ještě jednodušší)

1. Jděte na https://app.netlify.com/drop
2. Přetáhněte soubor `index.html` do okna
3. Web je online okamžitě!
